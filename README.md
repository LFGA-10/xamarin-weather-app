# WeatherFlow Premium 🌤️

A state-of-the-art, cross-platform weather application built with a focus on **Visual Excellence** and **High-Performance Logic**.

![Preview](https://images.unsplash.com/photo-1592210633464-a7db052983d3?auto=format&fit=crop&q=80&w=1000)

## ✨ Features
- **Real-time Weather**: Accurate weather data for any city worldwide.
- **Glassmorphic UI**: A stunning, modern design with smooth backdrop blurs and gradients.
- **Dynamic Context**: UI backgrounds and icons that change based on current weather conditions.
- **5-Day Forecast**: Stay ahead with a high-fidelity forecast visualization.
- **Premium Animations**: Powered by Framer Motion for a fluid, app-like experience.

## 🚀 Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Language**: TypeScript

## 🛠️ Installation
```bash
npm install
npm run dev
```

Originally a Xamarin skeleton, this project has been evolved into a modern Web/Mobile hybrid experience to meet the highest standards of the 2026 tech landscape.
